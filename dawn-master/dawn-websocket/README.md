# dawn-master

### dawn-websocket
WebSocket通信模块: Spring Boot 集成 WebSocket，实现后端主动往前端推送数据。网上大部分websocket的例子都是聊天室，本例主要是推送服务器状态信息。前端页面基于vue和element-ui实现。
